# LabCim Manager v7.2

Versão de ajuste visual e confirmação persistente de reserva.

## Correções desta versão

- Melhor contraste do banner principal.
- Botões com contraste reforçado, especialmente o botão de registrar reserva.
- Abas principais da página de reservas com destaque maior em formato de pílulas:
  - Calendário semanal
  - Calendário mensal
  - Agenda linear
  - Nova reserva
  - Gerenciar reservas
- Confirmação de reserva persistente após o `rerun` do Streamlit.
- Cartão de confirmação com equipamento, solicitante, período e número de amostras.
- Botão para ocultar a confirmação quando o usuário desejar.

## Como rodar

```powershell
pip install -r requirements.txt
streamlit run app.py
```
